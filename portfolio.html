TCC

Eu tive a oportunidade de fazer o meu TCC no BNDES. Realizei um projeto com ênfase na análise de dados para monitoração de acesso às bases do Data Lake por meio de um dashboard no Power BI. Tendo como objetivo mostrar todas as bases e os respectivos usuários que as acessaram em um período, além de mostrar o volume de dados dessas bases. O trabalho envolveu todas as etapas do ciclo de vida dos dados, desde o tratamento inicial dos dados de acesso, passando pela construção de consultas SQL até a visualização de dados por meio de relatórios e dashboards no Power BI.
